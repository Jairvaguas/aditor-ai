# Directiva: Rediseño de Landing Page (Parte 1)

## Objetivo
Modernizar la landing y sus componentes con un estilo premium, incluyendo animaciones y paleta de colores oscuros.

## Reglas
1. **Archivo global**: Actualizar `src/app/globals.css` para incluir las variables CSS de colores y las nuevas animaciones (fadeSlideUp, float, pulse-glow, scroll-left, shimmer).
2. **Navbar**: Reemplazar `src/components/Navbar.tsx` para modernizar UI, ajustando espaciados y el botón principal.
3. **Página Principal**: Reemplazar TODO el contenido de `src/app/page.tsx` por el nuevo diseño.
4. **Footer**: Reemplazar `src/components/Footer.tsx` para ajustarse al rediseño oscuro minimalista.
5. **Calidad**: Ejecutar `npm run build` para asegurar que el código typescript y css compila correctamente antes de pushear.

## Ejecución
1. Actualizar `src/app/globals.css`.
2. Actualizar `src/components/Navbar.tsx`.
3. Actualizar `src/app/page.tsx`.
4. Actualizar `src/components/Footer.tsx`.
5. Ejecutar commit & push en Git.
